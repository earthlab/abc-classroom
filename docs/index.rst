Welcome to abc-classroom's documentation!
=========================================

**abc-classroom** is a python package for course instructors that integrates
local development / marking of lesson material with distribution /collection
through `GitHub Classroom <https://classroom.github.com/>`_. It supports (but
does not require) `nbgrader <https://github.com/jupyter/nbgrader>`_ for managing
and autograding Jupyter notebooks.
